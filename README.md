# Focus Reset Stretch App

## Project Overview

Focus Reset Stretch App is a simple web-based prototype designed to reduce sedentary behaviour among students and working adults.

The idea is to encourage users to take short movement breaks after sitting for long periods. Users can scan a QR code or open the website, complete a short stretch activity using the camera, earn points, and view their score on a leaderboard.

This prototype was created for our UDE2222 project on reducing sedentary behaviour in study and work environments.

---

## Why We Created This Program

Many students and working adults sit for long hours while studying, attending classes, working, gaming, or using screens. Long periods of sitting can lead to discomfort, poor posture, stiffness, and reduced physical activity.

Our goal is to create a simple and low-effort solution that reminds users to move without disrupting their productivity.

This program supports the idea that short movement breaks can help users reset, stretch, and return to their work with better focus.

---

## Main User Flow

1. User opens the website or scans a QR code.
2. The app explains the stretch challenge.
3. User allows camera access.
4. User performs the required stretch or movement.
5. The app detects or confirms the stretch activity.
6. User earns points after completing the activity.
7. User can view their score on the leaderboard.
8. The leaderboard creates motivation through simple ranking and rewards.
